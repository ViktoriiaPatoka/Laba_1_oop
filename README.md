# Створення однозв'язного рекурсивного списку
14 варіант: 1, 3, 5, 12, 15, 20, 26, 40, 62, 76, 79
- Конструктор з одним параметром (число);
- Конструктор копіювання;
- Не рекурсивний метод додавання нового елемента останнім у
список;
- Метод додавання нового елементу після n-ного за рахунком
елемента;
- Рекурсивний метод видалення останнього в списку елемента;
- Рекурсивний метод видалення всіх елементів із заданим
значенням;
- Не рекурсивний метод друку елементів списку у прямому порядку
у рядок;
- Метод пошуку елемента із заданим значенням (результат -
посилання на знайдений елемент);
- Індексатор з двома параметрами, що дозволяє знайти мінімальне
серед значень із заданими індексами;
- Перевизначити для списку операцію явного перетворення до
типу string
- Перевизначити для списку будь-яку операцію(перевизначення операції "+", задля отримання з двох окремих списків - один)
